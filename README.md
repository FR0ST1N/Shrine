## How To Play

Don't Let the Tengu Faces Enter the Shrine.

### Controls

| Button  | Description |
| ------------- | ------------- |
| T | Toggle Control Info  |
| A  | Move Left  |
| D  | Move Right  |
| Left Click  | Shoot  |
| Space  | Jump  |
| Space + Space  | Double Jump  |
| R  | Restart Game (If Game Over)  |
